# Условные операторы, ветвление

1. Используя конструкцию if..else, напишите код, который получает значение prompt, а затем выводит alert:

- 1, если значение больше нуля,
- -1, если значение меньше нуля,
- 0, если значение равно нулю.

2. Перепишите с помощью тернарного оператора

```
if (a + b < 4) {
  result = 'Мало';
} else {
  result = 'Много';
}
```

3. А это с помощью *switch/case*

```
if (login == 'Вася') {
  message = 'Привет';
} else if (login == 'Директор') {
  message = 'Здравствуйте';
} else if (login == '') {
  message = 'Нет логина';
} else {
  message = '';
}
```

4. Проверка логина

Напишите код, который будет спрашивать логин (prompt).

Если посетитель вводит «Админ», то спрашивать пароль, если нажал отмена (escape) – выводить «Вход отменён», если вводит что-то другое – «Я вас не знаю».

Пароль проверять так. Если введён пароль «Чёрный Властелин», то выводить «Добро пожаловать!», иначе – «Пароль неверен», при отмене – «Вход отменён».
